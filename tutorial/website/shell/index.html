Shell工具介绍

更新时间：2024-6-26

简介

操作系统：可以分为 kernel (核心) 和 Shell (外壳) 两部分

shell是一个提供操作界面的软件，位于系统外层，用于连接系统内部核心

  • 图形界面shell：Windows系统桌面 explorer.exe

  • 命令行式shell：Windows系统的 cmd.exe

区别

shell说明
CMD是 Command shell 的简写，仅用于执行Windows系统命令和批处理文件
PowerShell是CMD的升级版，包含CMD所有功能且可跨平台使用

工具

连接服务器：终端(Xshell) → SSH协议 → shell

带 ⭐ 的是有教程的，点击即可跳转

shell客户端说明
CMDWindowsCommand shell 的简写，仅用于执行Windows系统命令和批处理文件
PowerShellWindowsCMD的升级版，包含CMD所有功能且支持脚本环境
OpenSSHWindows、 Mac、Linux是一款 免费开源 的SSH工具，它会加密客户端与服务器之间的流量，遏止窃听、劫持和攻击
⭐XshellWindowsNetSarang 公司旗下的一个安全终端模拟软件（SSH），教育版可免费使用
FinalShellWindows、 Mac、Linux一款国人开发的SSH客户端工具，支持shell和sftp同屏显示，文件管理等
MobaXtermWindows带有 X11 服务器、选项卡式 SSH 客户端、网络工具等的 Windows 增强终端，分为免费开源版和收费专业版
⭐TermiusiOS、Android、Windows、 Mac、Linux支持多平台的SSH客户端软件，可以实现SSH信息同步、FTP/SFTP连接、主题切换等功能
⭐JuiceSSHAndroid安卓端免费的SSH客户端，目前已停止更新
tabbyWindows、 Mac、Linux前身是 Terminus，是一个可高度配置的终端模拟器和 SSH 或串口客户端
WindTermWindows、 Mac、Linux开源 SSH 连接客户端，支持SSH/Sftp/Shell/Telnet/Serial terminal
electermWindows、 Mac、Linux免费开源的SSH客户端，支持ssh/sftp/telnet/serialport/RDP/VNC
SecureCRTiOS、Windows、 Mac、Linux一款支持SSH的终端仿真程序，同时支持Telnet和rlogin协议
kylinshellWindows一个简单易用的SSH和SFTP工具，提供免费版和商用版
PuTTYWindows最初由 Simon Tatham 为 Windows 平台开发的 开源软件
mRemoteNGWindows改进版开源 SSH 连接客户端，支持VNC、SSH、Telnet等多种协议，内置多种语言
